### 如何使用

**pip 安装**
```bash
$ pip install skr
```

**源码安装**
```bash
$ git clone https://github.com/chenjiandongx/skrskr.git
$ cd skrskr
$ pip install -r requirements.txt
$ python setup.py install
```

**Let's Skr**
```bash
$ skr
```
